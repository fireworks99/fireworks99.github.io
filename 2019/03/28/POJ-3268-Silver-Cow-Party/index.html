POJ 3268 Silver Cow Party

2019-03-28
  • spfa
  • 最短路
  • 题解
本文总阅读量次

Description

有n个农场（1~N），给出M条单向路描述。每个农场派出一头牛去某个农场（X）参加聚会，聚会结束后要返回，当然它们来回都选择最短路（因为路是单向的，来回可能路径不同），问这n头牛中，来回所走路程最长的那头，它走的总路程是多少?

题目链接 http://poj.org/problem?id=3268

思路

最短路，计算每头牛来回最短路之和，所有和中最大值即为答案
