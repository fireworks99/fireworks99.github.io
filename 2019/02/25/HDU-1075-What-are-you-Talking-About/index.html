HDU 1075 What are you Talking About

Description

Input

The problem has only one test case, the test case consists of two parts, the dictionary part and the book part. The dictionary part starts with a single line contains a string вҖңSTARTвҖқ, this string should be ignored, then some lines follow, each line contains two strings, the first one is a word in English, the second one is the corresponding word in MartianвҖҷs language. The book part starts with a single line contains a string вҖңSTARTвҖқ, this string should be ignored, then an article written in MartianвҖҷs language. You should translate the article into English with the dictionary. If you find the word in the dictionary you should translate it and write the new word into your translation, if you canвҖҷt find the word in the dictionary you do not have to translate it, and just copy the old word to your translation. Space(вҖҳ вҖҳ), tab(вҖҳ\tвҖҷ), enter(вҖҳ\nвҖҷ) and all the punctuation should not be translated. A line with a single string вҖңENDвҖқ indicates the end of the book part, and thatвҖҷs also the end of the input. All the words are in the lowercase, and each word will contain at most 10 characters, and each line will contain at most 3000 characters.

In this problem, you have to output the translation of the history book.

#include <map>
#include <cmath>
#include <string>
#include <iostream>
#include <algorithm>
using namespace std;

map<string, string> mp;
map<string, bool> vis;

int main()
{
    string s1, s2;
    cin >> s1;///еҝ…е®ҡжҳҜSTART
    while(1)
    {
        cin >> s1;
        if(s1 == "END")
            break;
        cin >> s2;
        mp[s2] = s1;
        vis[s2] = 1;
//        cout << "s1: " << s1 << '\n' << "s2: " << s2 << '\n';
    }
    cin >> s1;///еҝ…е®ҡжҳҜSTART
    getchar();///еҗҺйқўзҙ§и·ҹжңүgetline(иғҪиҜ»е…ҘеӣһиҪҰ)пјҢжӯӨеӨ„иҜ»еӣһиҪҰйҒҝе…Қй”ҷиҜҜ
    string tem;
    while(1)
    {
        getline(cin, tem);
        if(tem == "END")
            break;
        for(int i = 0, j = 0; i < tem.length(); i = j)
        {
            if(!(tem[i] >= 'a' && tem[i] <= 'z'))
            {
                cout << tem[i];
                j++;
            }
            else
            {
                while(tem[j] >= 'a' && tem[j] <= 'z')///жӢ“еұ•дёҠз•Ң
                    j++;
                string s = tem.substr(i, j - i);
                if(vis[s])
                    cout << mp[s];
                else
                    cout << s ;
            }
        }
        cout << '\n';
    }
    return 0;
}
