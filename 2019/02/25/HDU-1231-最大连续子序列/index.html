Description

з»ҷе®ҡKдёӘж•ҙж•°зҡ„еәҸеҲ—{ N1, N2, вҖҰ, NK }пјҢе…¶д»»ж„Ҹиҝһз»ӯеӯҗеәҸеҲ—еҸҜиЎЁзӨәдёә{ Ni, Ni+1, вҖҰ, Nj }пјҢе…¶дёӯ 1 <= i <= j <= KгҖӮжңҖеӨ§иҝһз»ӯеӯҗеәҸеҲ—жҳҜжүҖжңүиҝһз»ӯеӯҗеәҸеҲ—дёӯе…ғзҙ е’ҢжңҖеӨ§зҡ„дёҖдёӘпјҢ дҫӢеҰӮз»ҷе®ҡеәҸеҲ—{ -2, 11, -4, 13, -5, -2 }пјҢе…¶жңҖеӨ§иҝһз»ӯеӯҗеәҸеҲ—дёә{ 11, -4, 13 }пјҢжңҖеӨ§е’Ң дёә20гҖӮ еңЁд»Ҡе№ҙзҡ„ж•°жҚ®з»“жһ„иҖғеҚ·дёӯпјҢиҰҒжұӮзј–еҶҷзЁӢеәҸеҫ—еҲ°жңҖеӨ§е’ҢпјҢзҺ°еңЁеўһеҠ дёҖдёӘиҰҒжұӮпјҢеҚіиҝҳйңҖиҰҒиҫ“еҮәиҜҘ еӯҗеәҸеҲ—зҡ„з¬¬дёҖдёӘе’ҢжңҖеҗҺдёҖдёӘе…ғзҙ гҖӮ

Input

жөӢиҜ•иҫ“е…ҘеҢ…еҗ«иӢҘе№ІжөӢиҜ•з”ЁдҫӢпјҢжҜҸдёӘжөӢиҜ•з”ЁдҫӢеҚ 2иЎҢпјҢз¬¬1иЎҢз»ҷеҮәжӯЈж•ҙж•°K( < 10000 )пјҢз¬¬2иЎҢз»ҷеҮәKдёӘж•ҙж•°пјҢдёӯй—ҙз”Ёз©әж јеҲҶйҡ”гҖӮеҪ“Kдёә0ж—¶пјҢиҫ“е…Ҙз»“жқҹпјҢиҜҘз”ЁдҫӢдёҚиў«еӨ„зҗҶгҖӮ

Output

еҜ№жҜҸдёӘжөӢиҜ•з”ЁдҫӢпјҢеңЁ1иЎҢйҮҢиҫ“еҮәжңҖеӨ§е’ҢгҖҒжңҖеӨ§иҝһз»ӯеӯҗеәҸеҲ—зҡ„з¬¬дёҖдёӘе’ҢжңҖеҗҺдёҖдёӘе…ғ зҙ пјҢдёӯй—ҙз”Ёз©әж јеҲҶйҡ”гҖӮеҰӮжһңжңҖеӨ§иҝһз»ӯеӯҗеәҸеҲ—дёҚе”ҜдёҖпјҢеҲҷиҫ“еҮәеәҸеҸ·iе’ҢjжңҖе°Ҹзҡ„йӮЈдёӘпјҲеҰӮиҫ“е…Ҙж ·дҫӢзҡ„з¬¬2гҖҒ3з»„пјүгҖӮиӢҘжүҖжңүKдёӘе…ғзҙ йғҪжҳҜиҙҹж•°пјҢеҲҷе®ҡд№үе…¶жңҖеӨ§е’Ңдёә0пјҢиҫ“еҮәж•ҙдёӘеәҸеҲ—зҡ„йҰ–е°ҫе…ғзҙ гҖӮ
